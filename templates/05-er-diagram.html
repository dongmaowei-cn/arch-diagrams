7 实体·8 关系·10 元素图鉴

A 区 · 订单业务 ER 主图 — 电商核心域的 7 张表：users / orders / order_items / products / categories / payments / reviews。用 IE notation（crow's foot）表达 8 条关系，含 1—N / 1—1 / M—N (via junction) / 自引用（categories.parent_id）。

B 区 · 元素图鉴 — 10 张卡：实体三类（强 / 弱 / 关联表）+ 字段标记（PK / FK + [ex] UQ/IDX/NN）+ 4 种 crow's foot 端点 + 三种关系（1—N / M—N / 自引用）。主图实际使用的元素挂 olive ✓ 角标。

IE notation 的特点：表头是深色色块 + 字段以 zebra 行罗列 + PK/FK 用 inline tag 而非区域分隔 — 让阅读者一眼分清哪些字段是身份、哪些是引用、哪些是普通属性。
