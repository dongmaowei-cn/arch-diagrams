How to draw

垂直泳道的八步骨架

  1. 选方向：phase 横还是纵？此版 phase 纵向 (顶到底 = 时间)，lane 横向 (左到右 = 职能)。适合时间推进强、参与方少 (≤4) 的流程。
  2. 拉 phase column：左侧 60-80px 留给 phase 编号 + 名字 · phase 名 vertical-rl 竖排 · letter-spacing 加大呼吸感。
  3. 划 lanes：每 lane 一列 · 顶部 lane header 带 3px lane 色顶边 · 与下属节点同色形成视觉锚定。
  4. 节点按 lane 着色：customer = external gray · frontend = blue · service = olive · database = plum · 填充 8-10% · 描边同色 1.5px。
  5. 决策菱形特殊：用 clay 虚线菱形 与普通方形节点拉开 · YES 走 olive guard 边 · NO 走 rust fail 边。
  6. 边按类型选：spine olive 2.6px = happy path · slate 1.4px = 次要分支 · rust dashed = 失败 · plum dashed + 空心圆起点 = message flow (跨业务参与方)。
  7. phase divider：相邻 phase 之间画虚线 4 4 dash + 中点 clay ↓ · 强化时间推进感 · 不打断 lane divider。
  8. 失败终态：用 rust 粗双线圆 (stroke-width 2.5) + 圆内写状态码 (400 / 500) · 与 olive 起点细圆形成对仗。
